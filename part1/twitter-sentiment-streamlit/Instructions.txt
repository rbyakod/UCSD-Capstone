Docker Instructions:
====================

docker ps -a
docker rm <ID>
docker rmi <image ID>

docker build -t rbyakod/twitter-sentiment .
docker run -p 9000:9000 -d rbyakod/twitter-sentiment

docker login -u  rbyakod
docker push rbyakod/twitter-sentiment:latest
